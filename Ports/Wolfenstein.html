Wolfenstein 3D

Wolfenstein 3D é um jogo de tiro em primeira pessoa desenvolvido pela Id Software e lançado originalmente para o sistema MS-DOS em 1992. Mesmo não sendo o primeiro jogo de tiro em primeira pessoa (FPS) a existir, este sendo um chamado de "Maze War" lançado em 1974, Wolfenstein 3D foi a mais importante causa para a popularização e futuro desenvolvimento desse gênero de jogo, servindo como base para os jogos futuros (Ver Doom).

Wolf4SDL

O Wolf4SDL é uma família de ports refinados da engine original de Wolfenstein 3D feita para ser mais compatível com os hardwares modernos. Além das versões mais modernas do Wolf4SDL esse software é uma excelente forma de emular os jogos da série Wolfenstein 3D no RetroPie. Aliado ao Wolfenstein 3D, essa engine também dá suporte ao jogo "SPEAR OF DESTINY" que é uma sequência que se passa antes do próprio Wolfenstein 3D (prequel).
