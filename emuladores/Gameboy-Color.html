Gameboy Color

O Game Boy Color é o sucessor do Game Boy (Ver Game Boy) da Nintendo. Ele apresenta uma tela colorida, em vez de monocromática do modelo anterior e é ligeiramente mais grosso e alto do que o Game Boy Pocket. Assim como o Game Boy original, ele possui um processador de 8-bit

Lista de Jogos

lr-gambatte

O lr-gambatte é um dos emuladores de Game Boy Color disponíveis pela RetroArch (prefixo 'lr-') e é o único utilizado para a emulação de Game Boy Color no Ramo. Sendo assim, os comandos descritos na seção 'Comandos (RetroArch)' também são válidos para este emulador.
