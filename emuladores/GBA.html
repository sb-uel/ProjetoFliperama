Game Boy Advance

O Game Boy Advance (popularmente abreviado como GBA) é um console portátil desenvolvido e fabricado pela Nintendo. Lançado em 2001, é o sucessor do Game Boy Color (Ver Game Boy Color) e um dos últimos produtos da linha Game Boy. Seu codinome durante o processo de desenvolvimento foi Advance Game Boy. Ele continuou recebendo jogos até 2008.

Lista de Jogos

lr-mgba

O lr-mgba é um dos emuladores de GBA disponíveis pela RetroArch (prefixo 'lr-') e é o único utilizado para a emulação de GBA no Ramo. Sendo assim, os comandos descritos na seção 'Comandos (RetroArch)' também são válidos para este emulador.
