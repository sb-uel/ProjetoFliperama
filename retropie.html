RetroPie

Emulation Station

Emulation Station é um conjunto de emuladores open-source desenvolvido em C++ para, além de ser uma rede de diversos emuladores de jogos antigos, ser uma interface gráfica e temática. Para o RetroPie, este setor da emulação é responsável por fornecer a possibilidade de modificação das temáticas dos emuladores, como baixar as capas (cover) dos jogos, e reconhecimento e configuração dos joysticks.

RetroArch

RetroArch é uma estação de emuladores focado principalmente em diversificar a quantidade de plataformas emuláveis. Para o RetroPie, este sistem é responsável pela diversificação de emuladores e fornecimento de possivilidades de aprimoramento na emulação, como o aplicamento de 'shadders' para uma melhora gráfica.
